本项目所使用的 NoneBot 具有自带的 [插件广场](https://v2.nonebot.dev/plugin-store.html) <br>
当然前提是你已经安装`nonebot-cil`。
```shell
pip install nonebot2[cli]
poetry add nonebot2[cli]
```

如需导入第三方插件，请使用命令
```shell
 nb plugin 插件名
```
